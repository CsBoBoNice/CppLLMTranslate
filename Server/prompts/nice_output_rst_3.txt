以下是获取 ``i2c@40002000`` 节点节点标识符的几种方法：

- ``DT_PATH(soc, i2c_40002000)``
- ``DT_NODELABEL(i2c1)``
- ``DT_ALIAS(sensor_controller)``
- ``DT_INST(x, vnd_soc_i2c)`` 对于某些未知的数字 ``x`` 。有关详细信息，请参阅 :c:func:`DT_INST()` 文档。

.. important::

   设备树名称中的非字母数字字符，如连字符 ( ``-`` ) 和at符号 ( ``@`` ) ，将转换为下划线 ( ``_`` ) 。DTS中的名称也转换为小写。
